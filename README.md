# Video Chat Demo

**Your machine is already installed NodeJs.**

## Pre Requirements

```bash
  npm install -g ngrok
```

## How to run

```bash
  git clone <repo>
  cd <dir>
  npm install
  npm run dev
  ngrok http 5000
```

and then copy the **Forwarding URL** and paste in **scripts/index.js**.

### Open the Forwarding URL in browser.
